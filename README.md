# Tevhid.tr Web Sitesi

Kur'an ve sahih sünnet ışığında tevhid inancına davet eden, okuma odaklı ve minimalist web projesi. 

🌍 **Canlı Yayın:** [https://tevhid.tr](https://tevhid.tr)

## Proje Hakkında

Bu depo, `tevhid.tr` alan adında yayın yapan statik web sitesinin kaynak kodlarını barındırmaktadır. Ziyaretçilere kesintisiz, hızlı ve göz yormayan bir okuma deneyimi sunmak amacıyla tamamen statik mimari (HTML, CSS, JS) üzerine inşa edilmiş olup GitHub Pages üzerinden barındırılmaktadır.

## Temel Özellikler

* **Okuma Odaklı Tasarım:** Metinlerin rahat okunabilmesi için özenle seçilmiş tipografi (Inter ve Lora font aileleri) ve ideal satır boşlukları.
* **Gece/Gündüz Modu (Dark/Light Theme):** Ziyaretçilerin tercihine göre şekillenen ve tarayıcı hafızasında tutulan tema desteği.
* **Yüksek Performans:** Veritabanı veya ağır sunucu işlemleri içermeyen statik yapı sayesinde anında yüklenen sayfalar.
* **Mobil Uyumluluk:** Tüm cihazlarda (telefon, tablet, masaüstü) kusursuz çalışan esnek (responsive) ızgara sistemi.
* **SEO ve PWA Altyapısı:** Arama motorları için optimize edilmiş meta etiketleri ve çevrimdışı önbellekleme yetenekleri.

## Teknik Altyapı

* **HTML5:** Semantik ve erişilebilir sayfa iskeletleri.
* **CSS3:** Özelleştirilebilir renk paletleri, CSS değişkenleri (variables) ve modern animasyonlar.
* **Vanilla JS:** Tema değiştirici ve temel etkileşimler için dış kütüphanelere bağımlı olmayan saf JavaScript.
* **Hosting & CI/CD:** GitHub Pages altyapısı ile ücretsiz ve sürekli yayın.
